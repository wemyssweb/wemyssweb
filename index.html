Simply Responsive

The Goldilocks approach, design for the ideal size, then have a plan for smaller and bigger viewports. So web pages are based on ideal reading width, image sizes and design.

Simply easy to Navigate

You need to find what you want quickly. Easy to read and easy to know where you are by using common conventions, taking into account dissabilities.

Simply fast

We don't like to have to wait, and the more popular you become, the slower your website becomes. So I try to keep things fast though small file size, reducing processing overhead and caching.
